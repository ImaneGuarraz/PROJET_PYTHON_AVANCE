# 📘 Projets Python — Application Desktop & Analyse du Livre *Frankenstein*

Ce dépôt contient **deux projets Python distincts mais complémentaires**, réalisés dans le cadre de ton apprentissage du développement logiciel, de l’analyse de données et de la manipulation d’interfaces graphiques.

## 🖥️ Projet 1 — Application Desktop (Tkinter)

Le premier projet est une **application desktop complète**, développée avec **Tkinter**, permettant :

    - d’importer des fichiers
    - de les analyser
    - de visualiser des graphiques
    - d’interagir avec une interface moderne

> Pour découvrir l’architecture, le code et les fonctionnalités détaillées, rends‑toi dans le dossier :  

**`/projet_partie1/`**

## 📚 Projet 2 — Analyse du Livre *Frankenstein*

Le second projet est un pipeline d’analyse textuelle et visuelle autour du roman *Frankenstein* :

    - lecture du livre en local
    - extraction du chapitre 1
    - analyse des paragraphes
    - génération d’un graphique
    - traitement d’images
    - création d’un rapport Word final

> Pour explorer le code, les modules et les résultats générés, consulte le dossier :  

**`/projet_partie2/`**

---

Ces deux projets illustrent deux approches différentes :
- **une application graphique interactive**
- **un pipeline automatisé d’analyse et de génération de documents**

> Naviguer dans chaque dossier pour voir le code, les modules, les assets et les instructions spécifiques à chaque partie.
